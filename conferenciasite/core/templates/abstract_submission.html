Submission Guidelines

Content:

  • Title
  • Author’s name
  • Author’s affiliation (Institution, City, Country)
  • Keywords (Up to 6)
  • Main Body
  • Acknowledgments
  • Funding Information
  • References

Length:
Maximum 3 pages including all sections.

Format:

  • Title centered, bold, Times New Roman 14pt
  • Main text in 12pt TNR, 1.5 spacing
  • A4 page size, standard margins

Tables and Figures:

  • Only one per page
  • High-resolution

References:

  • APA style
  • Only those cited
